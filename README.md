## 📌 Overview
AuraView는 운전자 시야에서 직접 확인할 수 없는 **비가시 교통정보**를 분석하고,  
공공데이터와 비전 AI를 결합하여 **미래 위험 상황을 예측**하는 모바일 웹 서비스입니다.

단순 객체 인식을 넘어,  
**보이지 않는 정보와 공간을 계산하여 선제적으로 위험을 안내**합니다.

---

## 🔥 Pipeline
```

영상 입력 → 시야 차단 인식 → 비가시 영역 추정 → 공공데이터 연동 → Transformer 기반 위험 예측 → HUD 출력

```

---

## 🚀 Key Features

- 👁️ 시야 차단 인식 및 비가시 영역 추정
- 📡 공공데이터 기반 신호 정보 복원
- 🧠 Transformer 기반 시계열·시공간 위험 예측
- ⚠️ 보행자 / 차량 / 충돌 위험 사전 탐지
- 🧾 텍스트 기반 HUD 알림 제공
- 📊 위험 구간 데이터 축적 및 고도화

---

## 🎯 Use Cases

### 🚦 신호 가림 상황
전방 차량/구조물로 신호가 보이지 않는 경우  
→ `정지 12초`

---

### 🚶 보행자 출현 가능
횡단보도, 골목 등 위험 구간  
→ `보행자 출현 가능`

---

### 🚗 사각지대 접근 차량
측면/후방 사각지대 접근  
→ `충돌 위험`

---

## 🧠 Core Technology

### 1. 📊 비가시 교통정보 데이터화
- 시야 밖 정보(신호, 보행자, 차량 등)를 독립 데이터로 정의 및 축적

### 2. 📐 비가시 공간 추정
- 시야 차단 객체 기반으로 보이지 않는 영역을 계산

### 3. 🔮 Transformer 기반 위험 예측
- 위치, 속도, 시야 차단, 공공데이터, 이력 데이터 기반
- 미래 위험 발생 확률 예측

### 4. 🌐 크라우드 기반 고도화
- 사용자 데이터 축적 → 지역/시간별 위험 패턴 학습
