# AuraView

비가시 교통정보 축적·고도화 기반 지능형 안전 주행 지원 시스템

---

## 소개
AuraView는 운전자 시야에서 직접 확인하기 어려운 비가시 교통정보를 분석하고,
공공데이터와 비전 AI를 결합하여 위험 상황을 예측한 뒤
텍스트 기반 HUD 형태로 제공하는 모바일 웹 서비스입니다.

---

## 시스템 흐름
영상 입력 → 객체 탐지(비전 AI) → 공공데이터 연동 → 위험 예측 → HUD 출력

---

## 핵심 기능
- 전방 영상 기반 객체 탐지 (YOLO)
- 시야 차단 상황 인식 (비가시 영역)
- 공공 교통데이터 연동 (신호 정보 등)
- 위험 상황 예측 (머신러닝 기반)
- 텍스트 기반 HUD 알림 제공
- 위험 구간 시각화 (Dashboard)

---

## 주요 시나리오

1. 신호 가림 상황
- 전방 차량으로 신호등 가려짐
→ 정지 12초

2. 보행자 출현 가능
- 횡단보도 / 골목
→ 보행자 출현 가능

3. 사각지대 차량
- 측면 접근 차량
→ 충돌 위험

---

## 기술 스택
- Frontend: Mobile Web
- Backend: FastAPI 또는 Flask
- Vision AI: YOLO
- Prediction: LSTM / 머신러닝 모델
- Data:
  - 공공데이터 API (신호등)
  - GPS 위치 데이터
  - 사용자 주행 데이터

---

## 공공데이터 활용
- 교통안전 실시간 신호등 정보 API
- 교차로 위치 정보
- 도로 및 차로 정보
- 교통 흐름 정보

---

## 시스템 구조

[모바일 단말]
→ 영상 + 위치 수집
→ 비전 AI 분석
→ 공공데이터 연동
→ 서버 (데이터 축적)
→ 위험 예측 모델
→ HUD 출력
